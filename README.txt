
- The files and scripts contained within this folder are hosted and exceuted on Apocrita.
- To enable the full use of the mandarin app on a local machine you must first:
1) Set up Acces to apocrita using a private/public ID key (use ssh-copy-ID)
2) Adapt each file in this folder to reflect your apocrita username (change <username> to bt******)
3) Navigate to this folder in the terminal and run apocrita_setup.sh
4) Finally within the module.py script in the main app folder username must also be changed
